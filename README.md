# Arduino Smart Lock

This project was developed as a technological locking solution that integrates a keypad, a lock motor, and an LCD display to provide security and convenience for access control.

The lock is controlled through a numeric keypad, where the user must enter a code to open or close the lock. After the code is validated, the lock motor is automatically activated.

The lock status (open or closed) is displayed in real time on the LCD display, allowing the user to easily verify the system state.

The main goal of this project is to provide a simple and efficient access control solution using basic electronic components and straightforward programming.

This project is suitable for small security applications such as cabinets, small rooms, or as an educational electronics and microcontroller project.
